LimeLight Left (Blue) or Limelight Right (Yellow)

Pick Left or Right -> determines which side of the tree coral will dropped off at
Algae Yes or No? -> Will you need to clear an algae?  If so what level, 2 or 3? 
Coral Level -> 1 through 4


Blue Limelight
(1) If Left, Algae Yes, then accept coral input.
(2) If Right, Algae Yes, then coral input ignored.

(3) If Left, Algae No, then accept coral input.
(4) If Right, Algae No, then accept coral input.


Sequence:

(1) Move left X inches, move algae to correct level, move forward 2 inches, then run coral drop off at correct level.
(2) Move right X inches, move algae to correct level, move forward 2 inches.
(3) Move left X inches & move forward 2 inches, then run coral drop off at correct level.
(4) Move right X inches & move forward 2 inches, then run coral drop off at correct level.


Yellow Limelight
(5) If Left, Algae Yes, then ignore coral input.
(6) If Right, Algae Yes, then accept coral input.

(7) If Left, Algae No, then accept coral input.
(8) If Right, Algae No, then accept coral input.
